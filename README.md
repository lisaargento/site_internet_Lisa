# Site personnel – Lisa Argento

## 🎯 Objectif du projet

Ce site web a pour objectif de me présenter, de valoriser mes projets et de proposer un espace évolutif permettant de documenter mes travaux au fil du temps.

Il s’inscrit à la fois dans une démarche de développement personnel et professionnel, en me permettant de :
- présenter mon parcours et mes compétences
- mettre en avant mes projets (académiques, professionnels et personnels)
- expérimenter des outils et méthodes de développement web

Le site a été développé à partir d’une première version que j’avais réalisée, puis repensé et amélioré dans une logique de portfolio.



## 🛠️ Technologies utilisées

- HTML / CSS / JavaScript
- Framework CSS (à préciser : Bootstrap, Tailwind, etc.)
- GitHub pour la gestion de version
- Vercel pour le déploiement



## 🚀 Déploiement

Le projet a été développé en local, puis déployé en ligne afin de le rendre accessible.

Le site est ainsi désormais accessible publiquement via internet.
